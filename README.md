# security-research

Personal blog documenting a year-long deep dive into offensive security — web application testing, bug bounty, lab work, and original research.

**Live at [marieschmidt.me](https://marieschmidt.me)**

## What's here

- Blog posts covering PortSwigger Web Security Academy labs, HTB writeups, bug bounty reports, and tooling
- A live [PortSwigger progress tracker](https://marieschmidt.me/posts/postswigger-progress/) across all 146 labs
- Research notes, post-mortems, and lessons learned along the way

## Tech stack

Built with [Astro](https://astro.build) using the [Astro Chirpy](https://github.com/geocine/astro-chirpy) theme. Deployed to GitHub Pages with a custom domain via Cloudflare DNS.

## Local development

```bash
npm install
npm run dev
```

Site runs at `http://localhost:4321`.

## Build

```bash
npm run build
```
